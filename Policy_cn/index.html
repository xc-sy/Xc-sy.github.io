隐私政策

最后更新日期：2022年7月6日

1. 介绍

本隐私政策（以下简称“政策”）解释了xc-sy Game Studio（以下简称“xc-sy”，“我们”，“我们的”在本文中）如何收集、存储、使用或以其他方式处理我们服务的最终用户（以下简称“您”或“您的”在本文中）的个人数据，以及我们为何收集它、如何使用该信息以及您如何行使您的隐私权利。

在本政策中，“服务”指的是我们的游戏、网站以及我们直接控制的任何相关服务或属性。 “游戏”指的是我们拥有和运营的在线游戏或应用程序。请查看本政策适用的游戏列表。

通过安装、使用、注册或以其他方式访问我们的服务，您同意本隐私政策，并明确同意根据本隐私政策处理您的个人数据。如果您不同意本隐私政策，请不要安装、使用、注册或以其他方式访问我们的服务。

xc-sy保留定期修改和更新本隐私政策的权利，因此请定期查看此页面以审阅本政策。本隐私政策的更改将在更改上传到我们的网站后立即生效。您继续使用服务将表示您接受本隐私政策的更改。

2. 数据控制者

“数据控制者”是指确定如何以及为何处理个人数据的实体。您的个人数据将根据适用的数据保护法律由我们持有。根据数据保护法律的目的，数据控制者是xc-sy Game Studio。

在运营服务时，我们可能会与我们的合作伙伴共享您的数据。其中一些合作伙伴是独立于我们的数据控制者，因此他们独立决定如何以及为何处理您的数据。有关这些合作伙伴及其处理您的数据的方式的更多信息，请参阅本政策的第6节。

3. 我们处理的个人数据

当您访问我们的产品和服务时，我们不会收集您的法律敏感数据（例如种族、种族、宗教或哲学信仰、工会成员资格、遗传学、生物识别、健康状况、性取向和/或犯罪性）和其他敏感数据（例如儿童数据、社会安全号码、银行账户或支付卡详情）。请不要向我们提供此类信息或使用服务将其提供给他人。

我们可能会从您那里收集用户数据，包括：

  • 您提供给我们姓名、电子邮件地址或其他联系信息（例如，注册账户或订阅我们的通讯时）；
  • 您的在线标识符，如游戏中的别名、IP地址、我们分配给您的账户的内部标识符，或基于硬件或操作系统标识符，包括您的广告ID或设备ID；
  • 您使用我们的游戏或我们的其他服务时设备的 technical 信息（例如，其类型、型号、制造商、操作系统、语言、显示、处理器或内存）；
  • 从您的IP地址推断出的粗略位置（例如，国家、州或城市）；
  • 您使用服务的相关信息（例如，您在我们游戏中的进度信息、您访问我们网站的情况或与我们的游戏中的广告的互动）；
  • 您与我们的广告在服务之外互动的相关信息（例如，在第三方网站或应用程序上），包括您被引导到我们的游戏或其他服务的信息；
  • 与我们的服务相关的错误日志或其他与错误、错误或问题相关的信息；
  • 基于您在我们服务中的活动做出的推断；
  • 您通过服务发送的消息或其他通讯（例如，您在我们的游戏中的聊天消息或发送给我们的支持团队的电子邮件）；
  • 如果您选择将第三方账户（例如，社交媒体账户）与我们的服务连接，我们会从第三方账户提供商那里获取信息（例如，账户名称、标识符或电子邮件地址以启用跨设备游戏或提供社交功能），
  • 如果您在我们的游戏中进行购买，我们会从第三方支付服务提供商那里获取信息以验证这些购买（但不包括您的实际支付信息，如信用卡号码）；
  • 如果您参与我们的调查或其他研究，任何评论、反馈、回应或其他您提供的信息（例如，关于您的背景或我们游戏的经验）；
  • 您通过我们的服务或以其他方式选择提交给我们的任何其他信息；
  • 在您同意时，其他在您同意时解释给您的信息。

4. 我们如何收集您的数据

我们可能会通过各种方式收集和接收可能被视为个人数据的个人信息。更具体地说：

  • 当您向我们提交信息时直接从您那里收集（例如，注册我们的账户、订阅我们的通讯、向我们发送支持查询或回应我们的调查）；
  • 当您使用我们的服务或与服务之外的广告互动时，直接从您和/或您的设备通过自动手段收集信息（例如，通过cookies或类似技术或软件开发工具包（SDK）），
  • 从参与广告投放和测量的广告合作伙伴那里收集信息，如果您同意使用您的数据提供个性化广告（例如，在第三方网站或应用程序上）；
  • 如果您选择将第三方账户（例如，社交媒体账户）与我们的服务连接，从第三方账户提供商那里获取信息；
  • 如果您在我们的服务中购买，从参与这些交易的第三方支付服务提供商那里获取信息；
  • 在您同意时，从其他在您同意时解释给您的来源获取信息。

向我们要求数据不是强制的。但是，如果我们不处理您的数据，我们无法提供服务，或服务的某些部分或功能。如果您使用我们的服务，我们将根据您决定使用的服务和您使用它们时的选择，收集与您相关的数据。

您有多种选项可以限制或控制您的数据被处理的程度。例如，您可以选择不将您的第三方账户与我们的游戏连接，使用您的设备设置重置或限制您广告ID的使用，拒绝个性化广告的同意，或禁用浏览器设置中的某些或所有cookies。

6. 我们与谁共享您的数据

我们可能会与第三方共享您的数据，以实现本政策中描述的目的。这可能包括与以下类型的接收者共享数据：

  • 相关实体
  • 选定的商业合作伙伴和广告合作伙伴
  • 公共当局/监管机构/行业机构
  • 服务提供商
  • 替代服务提供商

如果xc-sy Game Studio或其附属公司的管理运营被转移给我们以外的公司，我们将披露我们持有的关于您的个人数据，以便产品和服务可以继续为您提供服务。

如果我们出售或购买任何业务资产，我们将披露我们的个人数据给潜在的买方或卖方，以便业务资产可以继续为您提供服务。

如果xc-sy Game Studio或其大部分资产被第三方收购，我们将披露我们持有的关于您的个人数据。

我们将根据收集数据的目的保留您的数据，包括提供您服务或遵守任何法律、会计或报告要求。我们可能会定期删除或去标识不活跃账户或其他数据在我们的游戏或其他服务中。在为安全执行服务所需的应用程序保留适用的保留期后，我们将删除或去标识您的数据，或者如果删除或去标识不可能（例如，由于数据存储在备份服务器上），我们将隔离您的数据，直到删除或去标识成为可能。我们可能继续使用无法与您关联的数据（例如，汇总数据）。在某些情况下，通过游戏可见的个人数据，如用户名、头像、最高得分和任何聊天消息，可能会缓存在其他玩家的设备上，我们可能无法从那些设备中删除或更新该数据，例如如果该设备未连接到网络。您可以使用下面描述的联系方式中的联系信息，询问特定情况和其他接收者。

我们非常重视您的个人数据的安全。我们实施适当的技术和物理安全措施，以确保您的个人数据免受意外或非法丢失、破坏、更改、未经授权的访问或披露，或任何其他未经授权的处理您的个人数据。我们用于个人数据的保护机制与其他游戏开发者类似。请记住，互联网不是100%安全的，由于我们保护措施的漏洞，通过我们的游戏访问、披露、更改或破坏您的个人数据可能无法保证。

我们的服务是全球性的，因此您的数据可以转移到世界任何地方。由于不同国家可能有不同的数据保护法律，我们采取措施确保本政策中解释的适当保护措施到位。这些包括实施欧洲委员会的标准合同条款，用于在集团公司和第三方服务提供商和合作伙伴之间转移个人数据，要求所有集团公司在根据欧盟数据保护法处理来自EEA的个人数据时保护个人数据。它还包括将个人数据转移到第三方服务提供商和合作伙伴，这些提供商根据欧盟-美国和瑞士-美国隐私盾被认证。

11. 用户数据删除

应用程序的用户有权管理、使用和删除应用程序生成的数据。如果用户不再需要应用程序中的账户数据，用户可以删除或删除它。

可删除的数据范围：1) 应用程序通过获取用户的基本Facebook账户信息；2) 用户账户登录记录；3) 应用程序中的所有用户操作记录，如等级记录、购买记录等。

在删除或删除用户之前，用户应了解的事项：根据用户的账户，删除或删除用户可能产生以下影响：1) 应用程序将删除所有用户数据；2) 在删除用户数据之前，您可能需要转移一些数据；3) 系统将不会删除某些数据，例如用户在Facebook上共享的应用程序相关内容；4) 删除用户数据后，将无法恢复。

删除应用程序中数据的方法：1) 前往您的Facebook个人资料，并在Facebook设置中找到应用程序和网站页面；2) 删除我们的应用程序；3) 点击已删除子选项卡，找到我们的应用程序，并发送删除数据请求。

12. 儿童政策

本服务不针对13岁以下的儿童。通过接受我们的服务条款并使用我们的服务，您表示您至少13岁。我们不会有意收集或征求13岁以下的个人数据，也不会允许此类人员使用我们的服务。如果您未满13岁，请勿使用该服务，也不要向我们发送任何关于您自己的信息。
